# AI Healthcare Assistant

## 📌 Project Description
This project is an AI-based healthcare assistant that helps users analyze their health using symptoms, medical reports, and image inputs.

## 🚀 Features
- Symptom-based disease prediction
- Medical report analysis
- Image-based detection
- Doctor recommendation system

## 🧠 Technologies Used
- Python
- TensorFlow
- SpeechRecognition
- CNN (Teachable Machine)
- Streamlit

## ⚙️ How It Works
1. User enters symptoms or uploads data
2. System analyzes input
3. Predicts possible disease
4. Recommends doctors

## 🎯 Objective
To provide quick and accessible healthcare guidance using AI.

## ⚠️ Note
This is a prototype and not a replacement for professional medical advice.
